# Income classification with bias and fairness analysis.

This predicts whether an individuals income is grater than $50K per year based in demographic and eployment attributes. Alongside model performance, teh project includes a bias and fairness analysis to evaluate whether predictions were influenced by sensitie attiributes such as gender.

The fairness evaluation uses metircs including **disparate impact, demographic parity** and **equal opportunity** to asssess whether the models behave differenty across gender groups

project Objectives
- Build a machine learning model to classify income levels
- Analyse model performance using standard evaluation metrics
- Address class imbalance using SMOTE
- Assess fairness across gender groups using quantitative fairness metrics
- Interpret whether the model exhibits bias and how this affects real world decision making
